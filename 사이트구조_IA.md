# UN R177 정보 웹사이트 — 구조(IA) 초안

> 수집자료를 토대로 한 사이트 정보구조 제안. 방향 확정 후 본격 구축.
> 샘플 프로토타입: `웹사이트/index.html` (단일 파일, 외부 의존성 없음).

## 1. 목표·대상
- **목표**: UN R177의 제정사·논의·시사점, 유럽 현황, 각국 대응·최신동향을 한곳에 정리한 레퍼런스 허브.
- **대상**: 정책·인증 담당(자안연/국토부), 완성차·부품사 규제담당, 연구자.
- **톤**: 공신력 있는 정부·연구 레퍼런스. 출처 링크 필수.

## 4. 기능 요소(권장)
- 상단 고정 내비 + 섹션 앵커 / 모바일 반응형.
- **타임라인 컴포넌트**(연혁), **비교표**(정렬·필터), **상태 배지**(확정/전망/미확인 ⚠️).
- 각 사실에 **출처 링크** 푸터.
- "최신 동향"은 추후 갱신이 잦으므로 **카드형 피드**로 분리.

## 5. 구축 옵션
- **A. 정적 단일 HTML**(현재 프로토타입): 배포 쉬움, 유지 단순. 소규모에 적합.
- **B. 정적 사이트 생성기**(예: MkDocs/Hugo): 다중 페이지·검색·문서중심에 적합.
- **C. 동적 대시보드**: '최신 동향'을 자동 갱신하려면 별도 데이터 연동 필요.
- 권고: 초기엔 **A 또는 B**로 콘텐츠 확정 → 동향 피드만 후속 자동화.
